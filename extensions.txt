1. Activities Configurator: Configure the Activities Button and Top Panel. Select an icon. Change the text.  
  https://extensions.gnome.org/extension/358/activities-configurator/
2. Block Caribou 36: Blocks caribou (the on screen keyboard) from popping up when you use a touchscreen.  
  https://extensions.gnome.org/extension/3222/block-caribou-36/
3. Dash to Dock: A dock for the Gnome Shell.  
  https://extensions.gnome.org/extension/307/dash-to-dock/
4. Fullscreen Hot Corner: Enables hot corner in fullscreen mode.  
  https://extensions.gnome.org/extension/1562/fullscreen-hot-corner/
5. OpenWeather: Weather extension to display weather information.  
  https://extensions.gnome.org/extension/750/openweather/
6. Quick Close in Overview: Close windows with a button click (the middle one by default) when in overview mode.  
  https://extensions.gnome.org/extension/352/middle-click-to-close-in-overview/
7. Remove Accessibility: Remove the accessibility button from the top panel.  
  https://extensions.gnome.org/extension/112/remove-accesibility/
8. Resource Monitor*: Monitor the use of system resources like cpu, ram etc.  
  https://extensions.gnome.org/extension/1634/resource-monitor/
9. Tray Icons: Reloaded: Tray Icons Reloaded is a GNOME Shell extension which bring back Tray Icons to top panel.  
  https://extensions.gnome.org/extension/2890/tray-icons-reloaded/
10. User Themes: Load shell themes from user directory.  
  https://extensions.gnome.org/extension/19/user-themes/
11. Snowy**: Adds snow to your system.  
  https://extensions.gnome.org/extension/3921/snowy/
  
* Merge `~/.local/share/gnome-shell-extensions/Resource_Monitor@Ory0n` with files from `resource-monitor-icons.zip` to change CPU and RAM icons.
** Extension made by ExposedCat. Thanks for testing and reviews on extensions.gnome.org.
